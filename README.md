# <p align="center"> QuoteCross </p>
QuoteCross is a unique word puzzle game that combines hidden quotes with interactive word discovery. Players solve clue words hidden throughout a grid, revealing an inspirational quote when all puzzles are completed.<p align="center">

</p>
Features
Core Gameplay

Interactive Grid: Tap letters to select and form words
Dynamic Puzzle Generation: Each game generates a unique grid layout with the same quote and clues
Word Detection: Automatically detects when you've selected the correct answer
Visual Feedback:

Yellow highlighting for selected letters
Grey coloring for solved words
Green highlighting for quote letters
Blank spaces for grid layout



Puzzle Progression

Multiple Puzzles: Start with 2 complete puzzles
Sequential Navigation: "Next Puzzle" button to progress through the game
Menu System: Easy access to puzzle selection and game reset

Gameplay Instructions

Select a puzzle from the main menu
View the clues in the clues panel
Tap letters in the grid to select them (they'll turn yellow)
Tap again to deselect if you make a mistake
Select all letters of a clue word in any order
When correct, the word is solved (letters turn grey)
Solve all clues to reveal the complete quote
View the revealed quote with proper formatting and attribution
Choose Menu to return to puzzle selection or Next Puzzle to continue

Technical Details

Language: Kotlin
Framework: Jetpack Compose
Min SDK: Android 24
Target SDK: Android 36
Version Code: 1

Known Limitations (Alpha)
Consider this an Alpha build that is just in the beginning stages of being built

Quote letters randomly placed  
No persistent progress saving
No difficulty levels
Limited hint system (meaning none)
